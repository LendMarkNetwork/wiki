About Lume

Welcome to Lume, the static site generator for Deno.

Lume is the Galician word for fire but also a static site generator for Deno.

It's inspired by other static site generators, such as Jekyll, Hugo or Eleventy, but it's easier to use and configure, besides being super flexible.

  • Supports multiple file formats, like markdown, yaml, JavaScript, typescript, jsx and nunjucks, and it's easy to extend with more.
  • You can hook processors to manipulate html and assets, like css or js.
  • It's Deno: Forget about managing thousands of packages in node_modules or complex bundlers. Lume only installs what you need. Clean, fast and secure.
